{% trans 'Ablauf' %}

Nachdem Sie den Marktmechanismus kennengelernt haben, beginnt nun das eigentliche Experiment. Im folgenden erhalten Sie alle relevanten Informationen für das Experiment. Lesen Sie sich diese bitte sorgfältig durch.

Das Experiment besteht insgesamt aus zwei vom Ablauf her identischen Sequenzen und einem anschließenden Fragebogen. Jede Sequenz besteht aus 12 Perioden. Jede Periode unterteilt sich wiederum in eine Handelsphase und eine Dividendenphase.

Nach Ablauf der ersten Sequenz erhalten Sie eine Übersicht über Ihr Vermögen, sowie neue Informationen für die zweite Sequenz.

In jeder Handelsphase können Sie am Kapitalmarkt Aktien von zwei verschiedenen Unternehmen handeln, um so Ihr Portfolio (= Aktienbestand) zu optimieren. In jeder Dividendenphase erhalten Sie anschließend für jede Aktie in Ihrem Portfolio eine Dividende (= Auszahlung). Um welche Unternehmen es sich handelt und wie hoch die Dividenden sind, erfahren Sie im Punkt Unternehmen und Dividenden.

Nachdem Sie die Instruktionen gelesen haben, müssen Sie einige Verständnisfragen beantworten.

{% trans 'Anfangsvermögen' %}

Zu Beginn jeder Sequenz erhalten Sie von uns ein Vermögen in Höhe von XX ECU. Zusätzlich erhalten Sie für Ihr Portfolio jeweils XX Aktien der beiden Unternehmen.

{% trans 'Ablauf' %}

Nachdem Sie den Marktmechanismus kennengelernt haben, beginnt nun das eigentliche Experiment. Im folgenden erhalten Sie alle relevanten Informationen für das Experiment. Lesen Sie sich diese bitte sorgfältig durch. Nur wenn Sie die Verständnisfragen am Ende der Instruktionen richtig beantwortet haben, wird das eigentliche Experiment gestartet.

Das Experiment besteht aus zwei Sequenzen und einem anschließenden Fragebogen. Eine Sequenz besteht aus 12 Perioden. Jede Periode unterteilt sich wiederum in eine Handelsphase und eine Dividendenphase.

In jeder Handelsphase können Sie am Kapitalmarkt Aktien von zwei verschiedenen Unternehmen handeln, um so Ihr Portfolio (= Aktienbestand) zu optimieren. In jeder Dividendenphase erhalten Sie anschließend für jede Aktie in Ihrem Portfolio eine Dividende (= Auszahlung). Um welche Unternehmen es sich handelt und wie hoch die Dividenden sind, erfahren Sie im Punkt Unternehmen und Dividenden.

Nachdem Sie die Instruktionen gelesen haben, müssen Sie einige Verständnisfragen beantworten.

{% trans 'Bankguthaben zu Beginn' %}

Zu Beginn jeder Sequenz erhalten Sie von uns ein anfängliches Bankverguthaben in Höhe von 1500 ECU. Zusätzlich erhalten Sie für Ihr Portfolio jeweils 5 Aktien der beiden Unternehmen.
